# Node Vue Template

A frontend template for vue development 

* Information
    * Vue 2 with CLI
    * TypeScript (Script)
    * Sass (Cascading Style Sheet)
    * Pug (Html Template)
    * Component with Class style
    * Fully modular

* Default dependencies
    * Vue 2
    * Vue-Router
    * Vuex
    * VueI18n
    * Axios
    * nprogress
